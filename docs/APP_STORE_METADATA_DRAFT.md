## Description

TRENOは、日々のトレーニングをシンプルに記録するためのアプリです。

カレンダーから日付を選び、その日のトレーニング内容を保存できます。部位、メモ、画像をまとめて記録できるので、前回の内容を振り返りながらトレーニングを続けやすくなります。

主な機能:

- 日付ごとのトレーニング記録
- 部位の入力
- メモの保存
- 画像添付
- カレンダー表示
- バックアップ書き出し・読み込み
- 端末内保存によるシンプルなデータ管理

大会やSNSのためではなく、自分の身体づくりを自分のペースで続けたい人向けのトレーニングログです。

## Privacy Summary Draft

現時点の想定:

- 外部サーバーへユーザーデータを送信しない
- 広告なし
- アナリティクスなし
- トレーニング記録・メモ・画像は端末内に保存

この実装が正しければ、App Store Connectのプライバシー回答は「Data Not Collected」寄りになる可能性が高い。
ただし、最終回答前にコードと利用SDKを確認すること。

## Age Rating Draft

## Review Notes Draft
